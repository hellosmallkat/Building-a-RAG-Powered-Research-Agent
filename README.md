## About This Repository

This repository contains my implementation of the National Student Data Corps (NSDC) Data Science Project, "Building a RAG Powered Research Agent."

The original educational project was created by the National Student Data Corps (NSDC), hosted by the Northeast Big Data Innovation Hub (NEBDHub). This repository includes both a blank student notebook and a completed reference notebook to support learning and self-study.

Original project:
https://nebigdatahub.org/nsdc-data-science-project-building-a-rag-powered-research-agent/# Building a RAG-Powered Research Agent

This repository contains a hands-on tutorial on building a Retrieval-Augmented Generation (RAG) research agent in Python. The project introduces the core concepts behind RAG systems and walks through implementing each component step by step.

The repository includes both a blank student version for learners to complete independently and a finished reference solution.

## Learning Objectives

By completing this project, you will learn how to:

* Understand the Retrieval Augmented Generation (RAG) pipeline
* Split documents into searchable chunks
* Generate vector embeddings
* Store and retrieve information using vector similarity search
* Build a simple research assistant that answers questions using retrieved context
* Evaluate the quality of a RAG system

## Project Overview

The notebook is organized into the following milestones:

### Introduction

Learn what Retrieval Augmented Generation is and why it is useful for AI applications.

### Milestone 1: Understanding RAG

Explore the architecture of a RAG system and answer reflection questions about each component.

### Milestone 2: Setup

Install the required libraries and prepare the development environment.

### Milestone 3: Building Components

Implement the major parts of a RAG pipeline, including:

* Document loading
* Text chunking
* Embedding generation
* Vector storage
* Similarity search

### Milestone 4: Running the Agent

Connect all components into a functioning research assistant capable of answering questions using retrieved documents.

### Milestone 5: Evaluating the System

Test the quality of the responses and analyze strengths and limitations of the RAG pipeline.

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/hellosmallkat/Building-a-RAG-Powered-Research-Agent.git

cd Building-a-RAG-Powered-Research-Agent
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

If a requirements file is not included, install the libraries used in the notebook manually.

### 3. Open the notebook

Launch Jupyter Notebook or JupyterLab.

```bash
jupyter notebook
```

or

```bash
jupyter lab
```
